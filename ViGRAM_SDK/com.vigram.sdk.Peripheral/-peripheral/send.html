abstract fun send(bytes: ByteArray, chunkSize: Int, callback: Callback1<DeviceMessage.Measurement>?)
Content copied to clipboard

Use this method to send out single messages to the viDoc. Messages to the peripheral are synchronized, so that even concurrent calls to this method will sequentially send out packets one message at a time.

  • Parameters:

  • data: The data to be sent to the viDoc. The data is sent unchanged (except for sending it in chunks of size chunkSize), so make sure to add padding and/or prefixes/suffixes beforehand.

  • chunkSize: The size to send individual packets in. It should not exceed `Configuration/maximumBLEPacketSize`.
